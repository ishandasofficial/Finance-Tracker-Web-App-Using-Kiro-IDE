💰 Finance Tracker Web App

A modern and user-friendly Finance Tracker Web Application built using Kiro IDE 🚀.
Track your income, expenses, savings, and manage your personal finances efficiently with a clean and responsive interface.
________________________________________________________

✨ Features

✅ Add and manage income & expenses
📊 Real-time balance calculation
📅 Track financial records easily
🎨 Clean and responsive UI
⚡ Fast and lightweight web app
🔒 Simple and user-friendly experience

________________________________________________________

🛠️ Tech Stack

🌐 HTML
🎨 CSS
⚡ JavaScript
🧠 Kiro IDE
